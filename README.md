# A multi-modal messaging app with accompanying automated UI tests

## Goal
The goal of this assignment is to design, implement and test a simple Android application. The proposed functionality of the application requires understanding and use of Activity class, UI design, event handlers and lifecycle callbacks, persistent storage, string analysis, and intent-based communication within and across applications.
